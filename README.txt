------------------Program Description-----------------
1. mandelbrot_serial.C
	Simple serial implementation for caculating Mandelbrot set

2. mandelbrot_omp.C
	Parallel implementation for calculating Mandelbrot set using OpenMP

3. mandelbrot_mpi_1
	Simple MPI implementation for calculating Mandelbrot set

4. mandelbrot_mpi_2
	MPI implementation for calculating Mandelbrot set using inter-communicators and intra-communicators

5. mandelbrot_mpi_3
	MPI implmentation for calculating Mandelbrot set using master-slave dynamic load distribution
------------------------------------------------------

-----------------Running the code---------------------
No command line arguments are required. timer.C needs to be compiled for the serial and OpenMP versions.
